Catálogo de Patrones de Desinformación

Análisis técnico de las estrategias estructurales utilizadas para propagar fake news en Venezuela. Este catálogo identifica firmas digitales y heurísticas de manipulación.

Supuesta Réplica Masiva

Distribución artificial y coordinada de contenido idéntico a través de múltiples canales en un corto periodo de tiempo para simular consenso.

INDICADORES TÉCNICOS

  • schedule Marcas de tiempo idénticas o secuenciales.
  • smart_toy Comportamiento automatizado (Bot-like behavior).

Patrón MULTIMEDIA Riesgo: Alta

Descontextualización Visual

Uso de material audiovisual real pero perteneciente a un evento, fecha o ubicación geográfica distinta para ilustrar una narrativa falsa.

INDICADORES TÉCNICOS

  • image_search Fotografías antiguas o de archivo.
  • info_i Metadatos alterados o eliminados (EXIF data).

Suplantación de Identidad

Clonación de elementos visuales de instituciones oficiales o medios de comunicación para otorgar credibilidad a documentos apócrifos.

INDICADORES TÉCNICOS

  • link Dominios visualmente similares (typosquatting).
  • text_fields Uso de tipografías no oficiales o logos deformados.
